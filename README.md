# Ecommerce Frontend

This is a frontend for an ecommerce application built with React, Redux, and Tailwind CSS.

## Features

- Product listing and details
- Shopping cart
- Wishlist
- User authentication
- Checkout process

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Technologies Used

- React
- Redux Toolkit
- React Router
- Tailwind CSS
- Vite
- Axios
- PostCSS
- ESLint
